使用说明

  1. 在下方 AI Agent 区域选择一个或多个 Agent
  2. 在每个 Agent 的提示词胶囊中点击 ✏️ 图标设置快捷键
  3. 通过胶囊右侧的开关 启用 该提示词的快捷键
  4. 在任何应用中按下快捷键即可触发对应 Agent 和提示词的 AI 分析

提示：快捷键无需打开本窗口即可生效。同一快捷键被多个提示词使用时，仅最先注册的有效。

⚙️ 配置

Auto insert: 自动粘贴到光标位置。Generate to clipboard: 复制到剪贴板，手动 Ctrl+V 粘贴。
SSE Fetch 通过 CDP 拦截 AI 接口的 HTTP 流式响应来获取回复，不需要窗口保持可见。DOM Poll 通过页面 DOM 轮询获取回复，需要豆包窗口保持打开状态。
Streaming 将 AI 回复按生成顺序逐段粘贴到目标应用。Full 等待完整回复后一次性粘贴。
