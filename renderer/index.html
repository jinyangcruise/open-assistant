🤖 Open Assistant

AI-powered code and document completion

使用说明

  1. 在下方 AI Agent 区域选择一个或多个 Agent
  2. 为每个 Agent 点击 录制按钮 设置快捷键
  3. 在任何应用中按下 Agent 的快捷键即可触发 AI 分析

提示：快捷键无需打开本窗口即可生效。

⚙️ Configuration

Auto insert: 自动粘贴到光标位置。Generate to clipboard: 复制到剪贴板，手动 Ctrl+V 粘贴。
SSE Fetch 在浏览器层面截获 AI 回复，窗口关闭到托盘时也能正常工作。DOM Poll 通过页面 DOM 轮询获取回复，需要窗口保持可见。
Streaming 将 AI 回复按生成顺序逐段粘贴到目标应用。Full 等待完整回复后一次性粘贴。
