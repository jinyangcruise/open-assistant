  1. 在首页 AI Agent 区域选择一个或多个 Agent。
  2. 点击右上角 ⚙️ 进入设置：管理提示词、配置通用选项
  3. 在任何应用中按下已启用的快捷键即可触发 AI 分析，结果会自动插入到光标位置

快捷键无需打开本窗口即可生效。同一快捷键被多个提示词使用时，仅最先注册的有效。

无法连接到豆包桌面端怎么办？
请点击 Agent 卡片上的 🔄 重启按钮，或在托盘菜单中点击「初始化豆包」，以正确调试端口启动豆包。同时检查端口是否被防火墙拦截。
快捷键无效怎么办？
检查是否与其他软件的快捷键冲突，尝试使用不同的快捷键组合。macOS 用户需要为应用授予辅助功能权限。
无法自动粘贴结果？
进入设置 → 通用 → 输出方式，确保选择了「自动粘贴到光标位置」。同时在等待回复的过程中确保正在使用的编辑器没有失去焦点。
